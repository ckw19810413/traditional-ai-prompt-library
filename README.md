> 高品質、經過驗證的繁體中文 AI Prompt 集合。不是 500 個通用 prompt 的雜亂合集，而是針對繁體中文使用者精心設計的 100 個高質量 prompt。

## 產品理念

市面上大多數 AI prompt 商品是「500 個通用 ChatGPT prompts 賣 $5」的廉價合集，抄襲嚴重、品質極差。**我們要做的完全不同**：

- 只有一個領域 → 繁體中文使用者
- 只有一個目標 → 高質量、即用即用的 AI 工作流
- 一個 prompt → 要能用，不是能看

## 為什麼我們的不同

| 項目 | 一般 prompt 商品 | 我們的 prompt 庫 |
|------|-----------------|-----------------|
| 語言 | 英文 | 繁體中文 |
| 品質 | 抄襲、通用型 | 原創、在地化 |
| 數量 | 500+ 但品質低落 | 只挑最好的 100 個 |
| 格式 | 純文字 | 有分類、有情境說明 |
| 更新 | 無 | 每月更新 10 個新 prompt |
