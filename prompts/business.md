# 商業應用 Prompt 集合

本文件包含 25 個精心設計的繁體中文商業 prompt，每個 prompt 都經過實測驗證。

---

## 一、商務郵件撰寫 (5 個 prompt)

### 1. 專業商務回覆郵件

```
你是一位有 10 年經驗的資深商務顧問。請幫我撰寫一封專業、得體的繁體中文商務郵件。

我的情境：{情境描述}
我的目的：{想達成的目標}
對方身分：{收件人身分/職稱}
語氣要求：{正式/半正式/友善}

請包含以下要素：
- 開頭問候語（符合商務禮儀）
- 說明來意（一兩句話）
- 核心內容（3 點以內）
- 行動呼籲（希望對方做什麼）
- 結尾客套話
- 專業結尾簽名

請直接給出完整郵件內容，不需要解釋。
```

### 2. 追問未回覆郵件的催辦信

```
請幫我寫一封追問郵件。對方已經超過 {天數} 天沒有回覆我的上一封信。

要求：
- 語氣要友善、不具攻擊性
- 假裝對方只是太忙忘了
- 簡短精煉（不超過 100 字）
- 给对方台階下（不施加壓力）
- 繁體中文

我的上封信主题是：{上封信主題}
我現在想詢問的是：{想詢問的內容}
```

### 3. 冷開發郵件 (Cold Email)

```
你是一位擅長冷開發的 B2B 銷售專家。請幫我寫一封有吸引力的冷開發邮件。

目標客戶：{對方公司/產業}
我賣的東西：{產品/服務描述}
我們的差異化優勢：{為什麼要選我們}
目標客戶可能有的痛點：{猜測的痛點}

要求：
- 標題要吸引人開啟（提供 3 個選項）
- 正文不超過 150 字
- 開頭要用「對方導向」而不是「我們導向」
- 有明確的 CTA（不要太多）
- 繁體中文
```

### 4. 會議後跟進郵件

```
請幫我寫一封會議後跟進郵件。

會議主題：{會議主題}
與會者：{對方身分}
我們討論的重點：{重點1、重點2、重點3}
我接下來要做的：{我的待辦事項}
我希望對方接下來做的：{對方的待辦事項}
下次聯繫時間：{時間}

要求：
- 包含會議摘要（3 個重點）
- 明確列出雙方待辦事項
- 建議下次聯繫時間
- 繁體中文，專業但不生硬
```

### 5. 客訴處理郵件

```
你是一位客戶成功專家。請幫我回覆一封客戶投訴郵件。

客戶的抱怨：{客戶說了什麼}
我們的錯：{事實真相，不要甩鍋}
我們能做的補償：{補償方案}
如何防止再發生：{改善措施}

要求：
- 先同理、再道歉、再解決方案
- 不要使用法律用語或防衛性語言
- 給出明確的補償和時間點
- 繁體中文
```

## 二、簡報大綱生成 (5 個 prompt)

### 6. 產品簡報大綱

```
請幫我生成一個產品簡報的大綱。

產品名稱：{產品名稱}
產品功能：{主要功能1、功能2、功能3}
目標受眾：{聽眾是什麼人}
簡報時長：{分鐘數}
核心訊息：{最想讓聽眾記住的一句話}

要求：
- 生成 10-15 頁的簡報大綱
- 每頁包含：標題、3 個重點、建議的視覺元素
- 開頭要有 Hook（抓住注意力）
- 結尾要有 Call to Action
- 繁體中文
```

### 7. 投資人簡報大綱

```
請幫我生成一個給投資人的簡報大綱。

公司名稱：{公司名}
正在募資的金額：{金額}
目前階段：{種子/A輪/B輪}
核心業務：{做什麼}
目前業績：{關鍵數據}
募資用途：{錢要怎麼花}

要求：
- 標準的 Pitch Deck 結構（10-12 頁）
- 每頁包含：標題、2-3 個重點
- 包含市場規模、解決方案、商業模式、競爭優勢、團隊、募資需求
- 繁體中文
```

### 8. 提案簡報大綱（對客戶）

```
請幫我生成一個商業提案簡報大綱。

客戶：{客戶公司}
提案內容：{我們要做什麼}
客戶目前的痛點：{問題是什麼}
我們的解決方案：{怎麼解決}
預期成果：{能帶來什麼好處}
報價：{價格}

要求：
- 以客戶痛點為開頭
- 重點放在「能為客戶帶來什麼價值」而不是「我們多優秀」
- 包含 3-5 頁的解決方案
- 包含案例研究
- 包含明確的下一步
- 繁體中文
```

### 9. 培訓簡報大綱

```
請幫我生成一個內部培訓簡報大綱。

培訓主題：{主題}
受訓對象：{聽眾是什麼職位/經驗水平}
培訓時長：{分鐘數}
期望受訓後能：{學會什麼}

要求：
- 由淺入深的結構
- 包含互動環節的設計
- 每頁包含：標題、重點、建議的講者筆記
- 包含一個實際練習
- 繁體中文
```

### 10. 季度業務報告簡報

```
請幫我生成一個季度業務報告簡報大綱。

季度：{Q1/Q2/Q3/Q4}
公司業務：{做什麼}
本季度重點指標：{關鍵數據}
本季度亮點：{成就}
本季度挑戰：{遇到的困難}
下季度計畫：{下一步}

要求：
- 數據導向的結構
- 包含 KPI 追蹤頁
- 包含問題分析和解決方案
- 包含明確的下季度目標
- 簡潔有力，不超過 12 頁
- 繁體中文
```

## 三、市場分析報告 (5 個 prompt)

### 11. 產業趨勢分析

```
請幫我生成一份 {產業名稱} 的趨勢分析報告。

關注時間範圍：{年份，如 2025-2027}
關注面向：{技術/市場/法規/消費者行為}

報告結構：
1. 執行摘要（3 個重點）
2. 產業現況
3. 三大趨勢（每個趨勢包含：是什麼、影響、機會/威脅）
4. 市場規模與增長預測
5. 主要參與者分析
6. 未來 12 個月的關鍵指標

要求：
- 用繁體中文
- 每點簡潔有力
- 包含具體的數字（如果有的話）
- 給出可執行的建議
```

### 12. 競品分析報告

```
請幫我分析 {我們的產品} 和 {競爭對手1}、{競爭對手2}、{競爭對手3} 的競爭格局。

分析面向：
- 價格
- 功能
- 目標客群
- 市場佔有率（估算）
- 優勢與劣勢

要求：
- 生成一個比較表格
- 列出我們的 3 個最大優勢
- 列出我們要提防的 3 個威脅
- 給出 3 個競爭策略建議
- 繁體中文
```

### 13. 目標客群分析

```
請幫我分析 {產品/服務} 的目標客群。

產品描述：{做什麼}
主要功能：{功能1、功能2}
定價：{價格範圍}

要求：
- 定義 3 個主要客群細分
- 每個細分包含：
  * 人口統計特徵
  * 痛點
  * 購買動機
  * 對價格的敏感度
  * 最可能使用的功能
- 給出針對每個細分的溝通策略
- 繁體中文
```

### 14. SWOT 分析

```
請幫我做一個 {公司/產品名稱} 的 SWOT 分析。

背景：{公司或產品的簡短描述}
主要優勢：{已知的優勢}
主要弱點：{已知的弱點}

要求：
- 列出 5 個 Strengths、5 個 Weaknesses、5 個 Opportunities、5 個 Threats
- 每個點簡潔有力（不超過 15 字）
- 給出 3 個 SO 策略（優勢×機會）
- 給出 3 個 WT 策略（弱點×威脅）
- 繁體中文
```

### 15. 市場進入策略分析

```
請幫我分析進入 {目標市場/國家} 的市場策略。

我們的公司：{做什麼、現有規模}
目標市場：{哪個市場}
目標客群：{誰會買}
法規限制：{知道的限制}

要求：
- 市場規模估算
- 主要競爭者
- 進入模式建議（直接銷售/合資/授權/電商）
- 前 6 個月的行動清單
- 預估成本結構
- 風險與對策
- 繁體中文
```

## 四、客戶回饋回應 (5 個 prompt)

### 16. 正面評價回覆

```
請幫我回覆一條正面評價。

產品/服務：{什麼}
客戶評價：{客戶說了什麼}
平台：{Google/Facebook/Instagram/官網}

要求：
- 真誠地感謝
- 提及客戶提到的具體優點
- 邀請下次再光臨
- 不超過 100 字
- 繁體中文，自然親切
```

### 17. 負面評價回覆

```
請幫我回覆一條負面評價。

產品/服務：{什麼}
客戶抱怨：{客戶說了什麼}
事實真相：{實際情況}
我們可以做的補救：{方案}

要求：
- 先同理、不辯解
- 承認問題（如果是我們的錯）
- 提出具體的補救方案
- 邀請私下聯繫
- 不超過 150 字
- 繁體中文，真誠不機械
```

### 18. 詢問價格的客戶回覆

```
請幫我回覆一條詢問價格的客戶訊息。

產品/服務：{什麼}
客戶問題：{客戶說了什麼}
我們的價格：{價格資訊}
我們的差異化：{為什麼值這個價}

要求：
- 不直接報價，先強調價值
- 簡述定價結構
- 提供一個低門檻的試用品項
- 有明確的 CTA
- 繁體中文
```

### 19. 要求退款的客戶回覆

```
請幫我回覆一個要求退款的客戶。

產品/服務：{什麼}
退款原因：{客戶說了什麼}
我們的退款政策：{政策內容}

要求：
- 先同理對方的不滿
- 說明政策但展現彈性
- 提供替代方案（升級/補償/教學）
- 如果符合條件則同意退款
- 保持良好關係
- 繁體中文
```

### 20. 推薦轉介的感謝信

```
請幫我寫一封感謝客戶推薦新客戶的郵件。

推薦的客戶：{名字/公司}
我們的產品：{做什麼}
推薦獎勵：{如果有}

要求：
- 真誠感謝
- 提及被推薦人的資訊（表明我們有跟進）
- 提及推薦獎勵（如果有）
- 簡短
- 繁體中文
```
