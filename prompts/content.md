# 內容創作 Prompt 集合

本文件包含 25 個精心設計的繁體中文內容創作 prompt。

---

## 一、社群貼文撰寫 (5 個 prompt)

### 1. Facebook 貼文

```
請幫我撰寫一篇 Facebook 貼文。

主題：{主題}
目標受眾：{誰會看}
產品/服務：{要推廣什麼}
關鍵訊息：{最想傳達什麼}
貼文目的：{引流/互動/品牌/銷售}

要求：
- 開頭要有 Hook（第一句就要抓住）
- 內文 200-400 字
- 包含 emoji 但不過度
- 有明確的 CTA
- 加上 3-5 個相關 hashtag
- 繁體中文，口語化但不随意
```

### 2. LinkedIn 專業貼文

```
請幫我撰寫一篇 LinkedIn 專業貼文。

主題：{主題}
目標受眾：{誰會看}
我的身分：{我的職位/專業領域}
想要建立的專業形象：{專家/實踐者/觀察者}

要求：
- 開頭用數據、問題或反直覺的觀點
- 內文 300-600 字
- 分段落，每段 2-3 句話
- 包含個人觀點或經驗分享
- 結尾引發討論
- 繁體中文，專業但不生硬
```

### 3. Instagram 貼文

```
請幫我撰寫一篇 Instagram 貼文。

產品/服務：{什麼}
目標受眾：{誰會看}
品牌風格：{專業/輕鬆/幽默/療癒}
想要的互動：{按讚/留言/分享/儲存}

要求：
- 開頭 1-2 句要夠吸引人
- 內文 100-250 字
- 分段清晰
- 有適當的 emoji
- 5-10 個 hashtag（混合熱門和小眾）
- 繁體中文
```

### 4. Twitter/X 貼文串

```
請幫我撰寫一篇 Twitter/X 的貼文串。

主題：{主題}
要傳達的核心觀點：{核心觀點}
目標受眾：{誰會看}
字數限制：{每則推文的字數}

要求：
- 第一則要有 Hook
- 包含 4-6 則推文
- 每則推文不超過 {字數} 字
- 包含數據或事實支持
- 包含個人觀點或經驗
- 最後一則有 CTA
- 繁體中文
```

### 5. TikTok 腳本

```
請幫我撰寫一個 TikTok 影片腳本。

主題：{主題}
目標受眾：{誰會看}
影片長度：{秒數}
產品/服務：{要推廣什麼，如果有的話}

要求：
- 前 3 秒要有 Hook
- 腳本格式：時間軸 + 畫面描述 + 旁白/對話
- 包含至少一個轉折或驚喜元素
- 結尾有明確的 CTA（關注/留言/分享）
- 繁體中文
- 口語化、年輕化
```

## 二、部落客文章 (5 個 prompt)

### 6. 教學型部落格文章

```
請幫我撰寫一篇教學型部落客文章。

主題：{主題}
目標讀者：{誰會看、知識水準}
文章長度：{字數}
目標關鍵字：{SEO 關鍵字}

要求：
- 標題要有吸引力且包含關鍵字
- 包含目錄
- 由淺入深的結構
- 每個步驟都要有具體範例
- 結尾有總結和下一步建議
- 繁體中文
- 適合 SEO 閱讀（短段落、小標題）
```

### 7. 產品評論文章

```
請幫我撰寫一篇產品評論文章。

產品名稱：{產品名}
產品類別：{什麼類型的產品}
價格：{價格}
主要優點：{3 個優點}
主要缺點：{2 個缺點}
目標讀者：{誰會看}

要求：
- 標題包含產品名和關鍵字
- 包含產品介紹
- 使用情境描述
- 詳細的優點和缺點分析
- 與同類產品的比較
- 明確的推薦/不推薦建議
- 繁體中文
```

### 8. 比較文章

```
請幫我撰寫一篇產品比較文章。

產品 A：{產品A名、主要特點、價格}
產品 B：{產品B名、主要特點、價格}
產品 C：{產品C名、主要特點、價格}
目標受眾：{誰會看}
使用情境：{什麼情況下用}

要求：
- 標題包含所有產品名
- 比較表格
- 各產品的優缺點
- 情境建議（什麼人選什麼）
- 不偏袒任何一方
- 繁體中文
```

### 9. 清單型文章

```
請幫我撰寫一篇清單型部落客文章。

主題：{主題}
清單數量：{幾個項目}
目標受眾：{誰會看}
關鍵字：{SEO 關鍵字}

要求：
- 標題格式「{數字} 個 {主題} 的方法/工具/技巧」
- 引言段：為什麼這份清單重要
- 每個項目：名稱 + 一句簡介 + 2-3 個重點
- 結尾：總結 + 建議選哪幾個
- 繁體中文
```

### 10. 案例研究文章

```
請幫我撰寫一篇案例研究文章。

公司/產品：{案例主角}
問題：{遇到了什麼問題}
解決方案：{怎麼解決的}
結果：{達成了什麼成果，有數據最好}
目標讀者：{誰會看}

要求：
- 標題要有趣且有數據
- 結構：背景 → 問題 → 解決過程 → 成果 → 啟示
- 包含具體的數字和數據
- 有引述或 Quotes
- 給出可應用的建議
- 繁體中文
```

## 三、產品文案 (5 個 prompt)

### 11. Landing Page 主文案

```
請幫我撰寫 Landing Page 的文案。

產品/服務：{做什麼}
目標客群：{誰會買}

差異化優勢：{為什麼選我們}

要求：
- Hero 區：標題 + 副標 + CTA
- 痛點區：列出具體痛點
- 解決方案區：我們怎麼解決
- 功能區：3 個核心功能
- 社交證明：客戶評價區
- FAQ 區：4 個常見問題
- 最後 CTA
- 繁體中文
```

## 四、電子報 (5 個 prompt)

### 16. 電子報首封

```
請幫我撰寫一封電子報首封。

電子報名稱：{名稱}

目標受眾：{誰會看}
主要內容：{要分享什麼}

要求：
- 標題要有吸引力
- 開頭要引人好奇
- 3 個主要內容（每個包含：標題 + 2 句話簡介 + 連結）
- 結尾有互動問題
- 繁體中文
```

### 17. 電子報每週更新

```
請幫我撰寫一封每週電子報。

這週分享內容：
1. {文章/資源標題 + 一句簡介}

3. {文章/資源標題 + 一句簡介}
4. {工具/應用推薦 + 一句簡介}

風格：{專業/輕鬆/幽默}
目標受眾：{誰會看}

要求：
- 主題行：有吸引力
- 開頭 1-2 句寒暄
- 每個內容項目包含：標題 + 2-3 句簡介 + 連結
- 結尾有個人感想
- 繁體中文
```

### 18. 促銷電子報


### 19. 再活化電子報

```
請幫我撰寫一封再活化（Re-engagement）電子報。

新的功能/內容：{最近有什麼更新}

要求：
- 標題要能引起注意（可以用問題）
- 承認很久沒聯絡
- 分享最近的更新
- 提供一個專屬的邀請或優惠
- 給他們一個「打開」的理由
- 繁體中文
```
